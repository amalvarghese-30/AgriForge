# AgriForge

A full-stack agricultural machinery e-commerce platform designed for farmers, dealers, and agricultural businesses.

## Features

- Product catalog with categories and brands
- Advanced product search and filtering
- Shopping cart and wishlist
- Secure authentication and authorization
- Order management and tracking
- Admin dashboard
- Inventory management
- Coupon and discount system
- Product reviews and ratings
- Razorpay payment integration
- MongoDB database
- Responsive design

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- TanStack Router
- Tailwind CSS
- ShadCN UI

### Backend
- Fastify
- Node.js
- MongoDB
- Mongoose
- JWT Authentication
- Cloudinary
- Razorpay
